# Statlige direktorater

Oversikt over statlige direktorater, etater og tilsyn i Norge.

## Instructions

Research and compile a comprehensive overview of Norwegian government directorates (statlige direktorater), including their names, responsible ministries, number of employees, and budgets.

## Result

- [Interactive overview](https://okms.github.io/research/statlige-direktorater/) (self-contained HTML)

## Datagrunnlag

**78 direktorater/etater** under **16 departementer**, validert mot:
- [Wikipedia: Liste over norske direktorat](https://no.wikipedia.org/wiki/Liste_over_norske_direktorat) (65 aktive)
- [DFØ: Organisering av virksomheter i staten](https://www.dfo.no/nokkeltall-og-statistikk/organisering-og-ansatte-i-staten/organisering-av-virksomheter-i-staten) (72 direktorater, 2025)

Lederverifisering per 17. april 2026: 44 av 78 lederposisjoner var utdaterte og er nå oppdaterte mot offisielle kilder (regjeringen.no, etatens egne ledelsessider). Se `notes.md` for detaljert liste over rettelser.

## KI-relevans

**15 etater** er vurdert som KI-relevante basert på offentlig tilgjengelige kilder:

| Etat | Begrunnelse |
|------|------------|
| NAV | Norges største offentlige KI-bruker – ML, chatbot, Copilot |
| Digdir | Etablerer KI Norge (aug. 2026), koordinerer 80/100%-mål |
| NSM | Risiko 2025 – KI som angrepsflate og forsvarsmiddel |
| Datatilsynet | Verdens første KI-sandkasse, EU AI Act |
| Skatteetaten | Forbilde iflg. Riksrevisjonen, generativ KI-plattform |
| Helsedirektoratet | KI for diagnostikk, e-helse, bildeanalyse |
| Politidirektoratet | Kripos KI-verktøy, digital etterforskning |
| Brønnøysundregistrene | Altinn 3, ML for svindeldeteksjon |
| Statens vegvesen | Autonome kjøretøy, trafikkflyt, digital tvilling |
| Kartverket | ML for satellittbilder og kartproduksjon |
| SSB | ML for klassifisering, NLP, alternative datakilder |
| Arkivverket | KI-OCR/HTR for historiske dokumenter |
| Miljødirektoratet | ML for fjernmåling, artsidentifisering |
| Tolletaten | KI for risikobasert grensekontroll |
| Nkom | Koordinerende tilsynsmyndighet for EU AI Act |
